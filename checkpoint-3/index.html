Cookie Clicker

Welcome to Cookie Clicker!

The object of this game is to get as high of a score as you can within 1 minute!

You gather points by clicking on the cookie, as well as while you play.

Points you receive can be spent on Multipliers to increase the amount of points per click.

Bonus Game: Minesweeper

This is a small Minesweeper-esque game I made. The object of this game is to get as high of a score as you can without hitting a bomb.

Click the squares to reveal a number between 1 and 3, and add it to your score.

Try to avoid to bombs, if you hit one it's Game Over!

Bonus Game #2: Memory Game

Here's another bonus game to try! This one is a Memory Matching game.

Click a square to reveal it's number and try to find it's matching number in the grid.

If you find all of the pairs, you win!
