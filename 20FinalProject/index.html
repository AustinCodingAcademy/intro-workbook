Portfolio Overview

This portfolio shows a few selections of my work. The projects are described below each image.

Project 1

Project 1 was my first major assignment to replicate a picture in the form of a webpage. I utilized flexbox with a parent container and two boxes.

Project 2

Project 2 was an assignment to replicate a picture of a professional portal for a company website designed for employers seeking employees as well as employees seeking employment. This assignment included an unordered list, section and div ids as well as an assortment of semantic html.

Project 3

Project 3 was an assignment that involved aligning three images with text beneath them as well as a button under each. This project also utilized flex-direction in order to align the images and text in a row.

Project 4

Project 4 was another major project. This one was a one-page website that included a navigation bar, table, unordered list, ordered list, with JavaScript to create smooth transitions through the site.
